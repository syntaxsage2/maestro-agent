# IntentRouter Chat 前端启动指南

## 📋 前置要求

1. **Node.js** - 版本 18.x 或更高
2. **npm** 或 **yarn** - 包管理器
3. **后端服务** - FastAPI 后端需要运行在 http://localhost:8000

## 🚀 快速启动

### 第一步：安装依赖

打开终端（PowerShell 或 CMD），进入项目目录：

```bash
cd F:\Python\agent_front
```

安装项目依赖：

```bash
npm install
```

这个过程可能需要几分钟，请耐心等待。

### 第二步：检查后端服务

确保 FastAPI 后端已经启动：

```bash
# 在另一个终端窗口
cd F:\Python\agent_Project2
python -m uvicorn intentrouter.api.main:app --reload --host 0.0.0.0 --port 8000
```

访问 http://localhost:8000/docs 确认后端正常运行。

### 第三步：启动前端开发服务器

```bash
npm run dev
```

看到类似输出表示启动成功：

```
  VITE v5.0.8  ready in 1234 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```

### 第四步：访问应用

打开浏览器，访问：http://localhost:5173

## 🎯 使用演示

### 1. 基础对话测试

1. 点击右上角"新对话"按钮
2. 输入：`你好，请介绍一下你自己`
3. 按 Enter 发送

### 2. 代码生成测试

输入：

2. 拖拽或选择一张图片
3. 输入：`这张图片里有什么？`

## 🔧 常见问题排查

### 问题 1：端口被占用

错误信息：`Port 5173 is already in use`

解决方法：
1. 找到占用进程并结束
2. 或修改端口：在 `vite.config.ts` 中修改 `server.port`

### 问题 2：无法连接后端

错误信息：`Network Error` 或 `网络错误，请检查后端服务是否启动`

检查清单：
- [ ] 后端服务是否运行在 http://localhost:8000
- [ ] 访问 http://localhost:8000/health 是否返回 JSON
- [ ] 检查 `.env` 文件中的 `VITE_API_BASE_URL` 配置

### 问题 3：依赖安装失败

如果 `npm install` 失败：

```bash
# 清除缓存
npm cache clean --force

# 删除 node_modules 和 package-lock.json
rm -rf node_modules package-lock.json

# 重新安装
npm install
```

或使用国内镜像：

```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### 问题 4：TypeScript 类型错误

运行类型检查：

```bash
npm run build
```

如有错误，根据提示修复。

- `react` & `react-dom` - React 框架
