====================================================================
所有可能的 MCP filesystem server 启动命令
====================================================================
直接复制到 Windows 终端（CMD 或 PowerShell）测试
按 Ctrl+C 停止运行的 server

--------------------------------------------------------------------
方案 1: 正斜杠路径
--------------------------------------------------------------------
npx -y @modelcontextprotocol/server-filesystem F:/Python/agent_Project2

--------------------------------------------------------------------
方案 2: 反斜杠路径（双反斜杠）
--------------------------------------------------------------------
npx -y @modelcontextprotocol/server-filesystem F:\\Python\\agent_Project2

--------------------------------------------------------------------
方案 3: 反斜杠路径（单反斜杠）
--------------------------------------------------------------------
npx -y @modelcontextprotocol/server-filesystem F:\Python\agent_Project2

--------------------------------------------------------------------
方案 4: 引号包裹（正斜杠）
--------------------------------------------------------------------
npx -y @modelcontextprotocol/server-filesystem "F:/Python/agent_Project2"

--------------------------------------------------------------------
方案 5: 引号包裹（反斜杠）
--------------------------------------------------------------------
npx -y @modelcontextprotocol/server-filesystem "F:\Python\agent_Project2"

--------------------------------------------------------------------
方案 6: 相对路径（从当前目录）
--------------------------------------------------------------------
cd F:\Python\agent_Project2
npx -y @modelcontextprotocol/server-filesystem .

--------------------------------------------------------------------
方案 7: 使用环境变量
--------------------------------------------------------------------
set PROJECT_DIR=F:\Python\agent_Project2
npx -y @modelcontextprotocol/server-filesystem %PROJECT_DIR%

--------------------------------------------------------------------
方案 8: PowerShell 变量（仅 PowerShell）
--------------------------------------------------------------------
$projectDir = "F:\Python\agent_Project2"
npx -y @modelcontextprotocol/server-filesystem $projectDir

--------------------------------------------------------------------
方案 9: 使用 npx.cmd 完整路径
--------------------------------------------------------------------
E:\nodejs\npx.cmd -y @modelcontextprotocol/server-filesystem F:/Python/agent_Project2

--------------------------------------------------------------------
方案 10: 通过 cmd /c
--------------------------------------------------------------------
cmd /c npx -y @modelcontextprotocol/server-filesystem F:/Python/agent_Project2

--------------------------------------------------------------------
方案 11: 混合盘符格式（WSL 风格）
--------------------------------------------------------------------
npx -y @modelcontextprotocol/server-filesystem /f/Python/agent_Project2

--------------------------------------------------------------------
方案 12: 短路径名（8.3 格式）
--------------------------------------------------------------------
先运行这个获取短路径：
dir /x F:\Python

然后使用短路径：
npx -y @modelcontextprotocol/server-filesystem F:\Python\AGENT~1

====================================================================
测试方法：
====================================================================

1. 打开 CMD 或 PowerShell
2. 逐个复制上面的命令测试
3. 如果 server 启动成功，你会看到它在等待输入（不会报错）
4. 按 Ctrl+C 停止
5. 把成功的命令告诉我

====================================================================
判断成功的标准：
====================================================================

? 失败：立即显示 "command not found: F:\Python\agent_Project2"
? 成功：server 保持运行状态，等待输入（没有错误信息）
